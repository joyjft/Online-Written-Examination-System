# Online-Written-Examination-System
A online based exam system made with HTML, servelet, jsp, apache server. All necessary files are in the zip file. You should use Netbean IDE to open this project.


Description :<br>
o	Platform : Web<br>
o	Language : Java,JSP<br>
o	Database : Java Database<br>
o	Core features : Login system for student,teacher and admin, teacher can create exam and can check script, student can perform exam and can see obtained mark<br>
